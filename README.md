# adventofcode2019

Source code for Advent of Code 2019 Chalenges.

## Requirements

* NodeJs 10+

## How to execute

1. Clone repository

1. Inside the project folder:

    ```javascript
        //Running Part 1 of Day 1 chalenge
        $ node day1/1.js

        //Running Part 2 of Day 1 chalenge
        $ node day1/2.js
    ```
